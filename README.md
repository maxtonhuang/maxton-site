# maxton-site

Personal portfolio site — plain HTML/CSS with a little vanilla JS. No frameworks, no build step.

Site files live in `public/`. On every push to `main`, GitHub Actions publishes the site to GitHub Pages.

## Local development

Open `public/index.html` directly in a browser, or serve the folder locally:

```bash
npx serve public
```

See [CLAUDE.md](CLAUDE.md) for project conventions.
